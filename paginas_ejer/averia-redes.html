No me puedo conectar a internet en Windows

• Verifica que el cable Ethernet esté correctamente conectado o que el adaptador Wi-Fi esté habilitado y conectado a la red correcta

• Reinicia el router y el ordenador para restablecer la conexión

• Asegúrate de que la configuración de red (IP, máscara de subred, puerta de enlace) esté correctamente configurada.

No me puedo conectar a internet en Linux

• Asegúrate de que la configuración de red esté correctamente configurada.

• Verifica si hay problemas con la conexión Wi-Fi o Ethernet y reinicia el router o el conmutador de red si es necesario

• Utiliza comandos como ifconfig o ip addr para verificar la configuración de red.

No se conecta a Internet

• Verifica que el cable Ethernet o la conexión Wi-Fi estén correctamente conectados

• Reinicia el router y el dispositivo afectado.

• Comprueba otros dispositivos en la red para determinar si el problema es específico de un dispositivo o de la red en general.

No se conecta al servidor

• Verifica la configuración de red y asegúrate de que el servidor esté disponible y accesible desde la red

• Comprueba la conexión y reinicia el servidor si es necesario

• Asegúrate de que no haya problemas de firewall o restricciones de acceso.

Puedo navegar con la IP pero no con la URL

• Verifica la configuración DNS en tu dispositivo y asegúrate de que esté configurada correctamente

• Intenta cambiar a servidores DNS alternativos como los de Google (8.8.8.8 y 8.8.4.4) para ver si resuelve el problema

Hay conexión a la red LAN pero no a Internet (WAN)

• Verifica la configuración del router y asegúrate de que esté correctamente configurado para proporcionar acceso a Internet.

• Comprueba si hay problemas con la configuración de la puerta de enlace predeterminada o con la conexión ISP

No puedo asociarme a un punto de acceso (AP)

• Asegúrate de que el SSID y la contraseña del punto de acceso sean correctos

• Reinicia el punto de acceso y el dispositivo cliente.

• Verifica si hay problemas de interferencia de señal o configuración de canal en el punto de acceso

No puedo acceder al router

• Verifica la dirección IP del router y asegúrate de que esté correctamente configurada en el navegador web

• Reinicia el router y comprueba si puedes acceder utilizando una conexión por cable en lugar de Wi-Fi.

En el switch, los LEDs no están de color verde ni parpadeando*

Esto puede indicar un problema con la conexión de red o un mal funcionamiento del switch. Verifica la conexión de red en los dispositivos conectados al switch y reinicia el switch si es necesario. Comprueba si hay problemas de configuración en el switch que puedan afectar su funcionamiento

La tarjeta de red tiene el LED de color rojo*

Esto puede indicar un problema con la conexión física, siendo normalmente un problema con el cable ethernet. Verifica si el problema persiste con otro cable ethernet
