¿Cómo puedo ver las extensiones de archivos en Windows?*

Para ver las extensiones de archivo te tienes que dirigir al explorador de archivos, en la opción de vista tiene que estar el check de “Extensiones de nombre de archivo” activado.

Tengo el disco duro casi lleno, ¿Cómo puedo liberar espacio?

Para liberar espacio puedes hacer los diferentes cambios:

- Eliminar archivos innecesarios
- Limpiar caché
- Eliminar aplicaciones no utilizadas
- Mover archivos a nube

Mejorar rendimiento del equipo

Para mejorar el rendimiento del sistema se pueden hacer algunas configuraciones.

- Desactivar los servicios innecesarios
- Deshabilitar aplicaciones de inicio
- Limpiar caché
- Reducir animaciones y efectos visuales
- Comprobar y corregir errores del sistema
- Optimizar configuración y energía

El sistema se congela o no responde de forma repetitiva.

Para solucionar estos congelamientos del sistema se puede hacer

- Realizar escaneo de virus
- Verificar temperatura
- Actualizar controladores
- Desactivar servicios
- Restauración del sistema

Windows tarda mucho en iniciarse*

Para solucionar esto se han de hacer diferentes cambios

- Desactivar programas de inicio
- Limpiar disco duro
- Desfragmentar el disco
- Verificar si hay un virus
- Actualizar drivers
- Desactivar servicios

Pantalla azul con error crítico

Primero se ha de revisar el código de error y verificar el problema, si el problema persiste, puedes hacer diferentes configuraciones:

- Intentar iniciar en modo seguro y restaurar el sistema a un punto de restauración anterior
- Reemplazar RAM o disco duro (Si es posible)
- Restauración del sistema

Error de arranque (Archivo de sistema faltante o corrupto)

Esto puede ser por un error en una actualización, un apagado incorrecto, o infección de malware. Solución:

- Recuperación de Windows
- Reparar los archivos con la opción reparación de inicio
- (si no funciona lo anterior) restaurar el sistema

Congelamiento en el inicio

Esto puede ser causado por cosas como: Sobrecalentamiento, fallos en la alimentación del pc, problemas de hardware. Solución:

- Iniciar en modo seguro y realizar un diagnóstico de hardware
- Herramienta para solucionar problemas de arranque de Windows
